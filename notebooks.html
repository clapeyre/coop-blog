Notebooks

Notebooks are useful material for self trainings. They are made of text and executable code boxes.

How to use them?

A public notebook has to be on a public Github repository. To use them, you must go on Nbviewer and paste the link to the notebook. On the new window, the notebook appears but the code boxes are not executable. Click on the Binter button on the top right of the window. On the new window, code boxes are executable.
